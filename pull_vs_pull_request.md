# Differences between Pull, Pull Request, and Push

| Action          | Description                                              | Direction                     | Purpose / When to use                          |
|-----------------|----------------------------------------------------------|-------------------------------|-----------------------------------------------|
| **push**        | Uploads your local commits to the remote repository      | Local → Remote (GitHub)       | Share your changes with others                 |
| **pull**        | Fetches changes from remote and merges them locally      | Remote → Local                | Update your local copy with others' changes    |
| **pull request**| Requests the repository owner to review and merge your changes from a branch/fork | Your repo/branch → Original repo | Propose and discuss changes before merging    |

**Summary:**
- `push`: Send your work up
- `pull`: Bring others' work down
- `pull request`: Ask permission to add your work to the main project

Sources:
